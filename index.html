🔥 Connect to Firebase

To make the system live with real-time sync across all devices, connect your Firebase project. Your data will be stored in Firebase Realtime Database and stay updated instantly.

1
Go to console.firebase.google.com → create a project (e.g. alif-furniture)
2
Click Build → Realtime Database → Create database → choose a region → start in test mode
3
Go to Project Settings (⚙) → General → scroll to "Your apps" → click the </> web icon → register app → copy the firebaseConfig values below
4
Paste each value from your config and click Connect & Launch. Config is saved locally so you only do this once.

Config is saved in browser localStorage. To reset, open browser console and run localStorage.removeItem('alif_fb_config')
